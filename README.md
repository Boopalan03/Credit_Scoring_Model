# Credit_Scoring_Model

AI-Powered Credit Scoring & Loan Eligibility System is a web application that uses Machine Learning to assess an applicant's creditworthiness based on financial details such as income, expenses, savings, and EMIs. It predicts loan eligibility, generates a credit score, explains decisions, detects fraud, and includes an AI chatbot for user guidance.

## 🚀 Live Deployments

* **Frontend Application (Vercel)**: [https://credit-scoring-model-three.vercel.app](https://credit-scoring-model-three.vercel.app)
* **Backend API (Render)**: [https://credit-score-backend-8hrn.onrender.com](https://credit-score-backend-8hrn.onrender.com)
